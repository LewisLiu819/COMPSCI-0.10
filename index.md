# COMPSCI 0.10 (CS 0.10): 小学计算机科学课程
*又称 CS 010 / COMPSCI 010 - 改编自加州大学伯克利分校 CS10 课程*

## CS 0.10 课程概述

欢迎来到 **COMPSCI 0.10**（也称为 **CS 0.10**、**CS 010** 或 **COMPSCI 010**）！这是一门专为小学4-6年级学生设计的计算机科学入门课程。

| 课程信息 | 详情 |
|---------|------|
| **课程名称** | COMPSCI 0.10 / CS 0.10 / CS 010 |
| **课程时长** | 16周（32节课，每周两节） |
| **每节时长** | 50分钟 |
| **适用年级** | 4-6年级（9-12岁） |
| **班级规模** | 15-20名学生 |
| **先修要求** | 无需任何编程基础 |
| **课程费用** | 免费 |

### COMPSCI 0.10 学习内容

在这门 CS 0.10 课程中，学生将学习：
- **编程基础** - 使用 Scratch Jr. 进行可视化编程
- **计算思维** - 培养逻辑推理和问题解决能力
- **算法设计** - 学习分步骤思考和指令编写
- **数字公民素养** - 了解网络安全和负责任的技术使用
- **创意计算** - 通过游戏和故事项目激发创造力
