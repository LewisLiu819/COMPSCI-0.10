# 课程概述

小学计算机科学："编程探险" - *改编自加州大学伯克利分校CS10，适用于4-6年级*

## 课程结构

### [单元1：什么是计算机科学？](week01/)（第1-2周）
- [第1周](week01/) - 计算机和数字世界介绍
- [第2周](week02/) - 像计算机一样思考和算法

### [单元2：编程基础构建](week03/)（第3-5周）
- [第3周](week03/) - Scratch Jr.介绍和基础编程
- [第4周](week04/) - 条件语句和交互式编程
- [第5周](week05/) - 循环和模式

### [单元3：问题解决和逻辑](week06/)（第6-8周）
- [第6周](week06/) - 布尔逻辑和决策制定
- [第7周](week07/) - 调试和结对编程
- [第8周](week08/) - 数据组织和数字项目

### [单元4：创意计算](week09/)（第9-11周）
- [第9周](week09/) - 数字故事讲述和角色发展
- [第10周](week10/) - 游戏设计和交互体验
- [第11周](week11/) - 数字艺术和音乐编程

### [单元5：计算机与社会](week12/)（第12-13周）
- [第12周](week12/) - 技术职业和社会影响
- [第13周](week13/) - 数字公民和全球连接

### [最终项目和评估](week14/)（第14-16周）
- [第14周](week14/) - 毕业设计项目规划和开发
- [第15周](week15/) - 项目完成和展示
- [第16周](week16/) - 展示活动和课程反思

## 课程详情

**学期长度：** 16周（32节课，每周2节）
**单节课时：** 50分钟
**目标年龄：** 4-6年级（9-12岁）
**班级规模：** 15-20名学生
**先修要求：** 无

## 学习目标

在本课程结束时，学生将能够：

- 理解计算机科学的基本概念，包括算法、编程和计算思维
- 使用Scratch Jr.等可视化编程工具创建交互式程序
- 通过编程挑战和调试练习培养问题解决技能
- 设计和实现创意项目，包括游戏、故事和数字艺术
- 理解技术在社会中的作用并实践负责任的数字公民行为
- 在编程项目上有效合作并提供建设性反馈
- 自信地展示他们的作品并解释他们的编程选择

## 评估理念

本课程强调成长、创造力和概念理解，而非技术完美。学生的评估基于：

- **过程重于结果：** 关注学习过程和进步
- **创造力和创新：** 鼓励独特的解决方案和创意表达
- **合作：** 与同伴有效合作并提供支持性反馈
- **坚持：** 拥抱挑战并从错误中学习
- **沟通：** 清楚地解释想法并自信地展示作品
